# Privacy Policy

Last updated: 2026-02-25

## GitHub Blind Reviews

GitHub Blind Reviews is a browser extension that anonymizes identities on GitHub pages to help reduce review bias.

## Data Collection

This extension does **not** collect, transmit, sell, or share personal data.

## Data Storage

The extension uses the browser `storage` permission only to store one local setting:

- `blindReviewsEnabled` (boolean): whether anonymization is enabled or disabled.

This setting is used only to control extension behavior in your browser.

## Host Permissions

The extension requests access to:

- `https://github.com/*`

This access is used only to apply anonymization on GitHub pages.

## Third Parties

The extension does not use analytics, advertising SDKs, tracking pixels, or external data processors.

## Remote Code

The extension does not execute remote code.

## Contact

For questions about this policy, open an issue in this repository.
